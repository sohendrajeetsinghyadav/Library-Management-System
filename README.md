📚 Library Management System (C)

A console-based Library Management System built using C that allows efficient management of books, members, and transactions with file handling for data persistence.


-----------------------------------------------------------------------------------

🚀 Features

📖 Add, view, search, and delete books

👤 Manage library members (add/view)

🔄 Issue and return books

💰 Fine calculation for late returns

💾 File handling for storing records permanently

🎨 Colored terminal interface for better user experience



-----------------------------------------------------------------------------------

🛠️ Tech Stack

Language: C

Concepts Used:

Structures

File Handling (fopen, fread, fwrite)

Arrays & Strings

Functions (Modular Programming)




-----------------------------------------------------------------------------------

📂 Project Structure

Library-Management-System/
│
├── main.c              # Main source code
├── books.dat           # Book records (binary file)
├── members.dat         # Member records (binary file)
├── transactions.dat    # Issue/return records
└── README.md

▶️ How to Run

1. Clone the repository:



git clone https://github.com/sohendrajeetsinghyadav/Library-Management-System.git

2. Navigate to the project folder:



cd Library-Management-System

3. Compile the code:



gcc main.c -o library

4. Run the program:



./library
